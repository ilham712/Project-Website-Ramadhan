# 🌙 Ramadhan Portal

Website interaktif untuk menemani ibadah di bulan Ramadhan. Dibuat dengan HTML, CSS, dan JavaScript murni (vanilla).

## 🚀 Fitur Utama

### 1. **Doa Harian**
- Doa sahur dan buka puasa
- Teks Arab, latin, dan arti

### 2. **Hitung Zikir**
- Counter interaktif
- Target 33 kali
- Subhanallah, Alhamdulillah, Allahu Akbar

### 3. **Kalkulator Zakat**
- Zakat penghasilan
- Zakat emas
- Perhitungan otomatis sesuai nisab (85 gram)

### 4. **To-Do List Ramadhan**
- Checklist shalat 5 waktu
- Tracker baca Qur'an
- Kalender puasa 1-30 Ramadhan
- Progress tracking visual

### 5. **Jadwal Imsakiyah**
- Jadwal sholat per kota
- Data real-time dari API MyQuran
- Highlight hari ini

## 📱 Cara Menggunakan

1. **Buka Website**
   - Kunjungi: [URL website kamu nanti]
   
2. **Navigasi**
   - Gunakan menu navbar untuk berpindah halaman
   
3. **Fitur Offline**
   - Beberapa fitur (Doa, Zikir, To-Do) bisa digunakan tanpa internet
   - Jadwal Sholat memerlukan koneksi internet

## 🛠️ Teknologi

- HTML5 (Semantic)
- CSS3 (Flexbox, Grid, Responsive)
- JavaScript (Vanilla ES6+)
- API: MyQuran API
